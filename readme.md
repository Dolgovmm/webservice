Веб-сервис - часть программного комплекса аналитики веб-сайтов, созданного в рамках обучающей стажировки портала GeekBrain.

Задача нашего проекта:

По указанным ключевым словам проанализировать заданный веб-ресурс
Собрать статистику
Предоставить статистику пользователю за необходимый период времени

Веб-сервис транслирует REST запросы от Мобильного и Десктопного интерфейсов пользователя, в запросы к базе данных. 
Результат, полученный от запроса, передает обратно Мобильному и Десктопному интерфейсу.
